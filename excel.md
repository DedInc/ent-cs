# MS Excel

## I. Общие сведения и назначение

**MS Excel** — прикладная программа из пакета Microsoft Office, представляющая собой среду обработки числовой информации. Работа приложения схожа с работой калькулятора, но предоставляет значительно большие возможности обработки числовых данных.

### 1.1. Основные характеристики

| Характеристика | Описание |
| :--- | :--- |
| **Основной файл** | Называется **Книгой** (рабочей книгой) |
| **Расширение файла** | **.xls** или **.xlsx** |
| **Структура Книги** | Состоит из нескольких **листов** (по умолчанию — три). Одна книга может содержать до 255 листов |
| **Основной элемент** | **Ячейка** — прямоугольник на пересечении столбцов и строк |
| **Адрес ячейки** | Составляется из буквенного имени столбца и номера строки (например, A1, B105) |
| **Размеры листа (2003)** | 255 столбцов и 65 535 строк |
| **Размеры листа (2007+)** | **16 384 столбца** и **1 048 576 строк** |

### 1.2. Области применения

- Подготовка бухгалтерских, экономических, статистических отчетов
- Решение задач разной сложности
- Построение диаграмм и графиков
- Ведение семейного бюджета
- Создание баз данных
- Математическое и теоретическое моделирование

### 1.3. Отличия от СУБД

- Документ в электронной таблице предназначен для выполнения **только одной определенной задачи**
- Электронные таблицы **непригодны для совместного пользования**
- Функций электронных таблиц **недостаточно** для осуществления полного комплексного мониторинга информации
- В реляционной БД поля (столбцы) строятся из элементов **одного типа**, в Excel это не обязательно

## II. Запуск и структура окна

### 2.1. Способы запуска приложения MS Excel

1. **Пуск** → **Программы** → **Microsoft Office** → **Microsoft Excel**
2. Двойной щелчок левой кнопки мыши по ярлыку на Рабочем столе
3. Однократный щелчок по пиктограмме приложения в панели задач

### 2.2. Основные части окна MS Excel

- **Первая строка:** Заголовок окна и кнопки управления
- **Вторая строка:** Строка меню приложения
- **Третья строка:** Лента (включает Стандартную, Форматирование и другие панели инструментов)
- **Четвертая строка:** Поле имени (адрес активной ячейки) или строка формул
- **Рабочая область:** С пятой строки до конца окна
- **Строка состояния:** В нижней части окна (индикаторы рабочих режимов)
- **Табличный курсор:** Прямоугольник, который показывает текущую (активную) ячейку
- **Диапазон ячеек:** Группа из нескольких ячеек

## III. Типы данных и ввод

### 3.1. Типы данных в ячейках

| Тип данных | Описание | Выравнивание | Особенности |
| :--- | :--- | :--- | :--- |
| **Число** | Целые, десятичные дроби, дробь. Могут включать $, %, +, –, /. Форматы: общий, финансовый, дата, время, процентный, дробный, текстовый | По левому краю | Если число не вмещается, ячейка заполняется символами **########** |
| **Текст** | Алфавитно-числовая информация (если запись не является числом или формулой). Используется для заголовков, пометок | По правому краю | При вводе строка начинается с букв или апострофа (́) |
| **Формула** | Запись должна начинаться со знака **=**. Например: `=А1 * В1` | - | При вводе значения формулы выводятся результаты вычислений |

### 3.2. Ввод данных

**Ввод данных** в ячейку осуществляется:
- Нажатием на клавишу `Enter`
- Щелчком по кнопке ввода ✓ в строке формул

## IV. Операции и форматирование

### 4.1. Форматирование ячеек

**Форматирование ячеек** осуществляется через диалоговое окно **Формат ячейки** (вызывается правой кнопкой мыши).

#### Вкладки окна «Формат ячейки»

- **Число:** Назначение формата для содержимого ячейки (числовой, денежный, дата и т.д.)
- **Выравнивание:** Выравнивание данных (по краю, центру), перенос слов, объединение ячеек, изменение направления текста
- **Шрифт:** Изменение цвета, размера, типа текста (надстрочный/подстрочный знак)
- **Граница:** Установка границ ячеек. Если границы не установлены, при распечатке они не отображаются
- **Заливка:** Изменение цвета одной ячейки или целой таблицы
- **Защита:** Защита данных в ячейках от удаления или изменения (действует после защиты листа)

### 4.2. Операции с ячейками

- **Удаление данных:** Выделить ячейки и нажать клавишу `Delete` или команду **Очистить** на Ленте
- **Копирование:** Кнопка **Копировать** или `CTRL+C`
- **Вырезание:** Кнопка **Вырезать** или `СTRL+X`
- **Вставка:** Кнопка **Вставить** или `CTRL+V`

### 4.3. Адресация ячеек

| Вид ссылки | Описание | Признак | Применение |
| :--- | :--- | :--- | :--- |
| **Относительная** (Relative reference) | Используется по умолчанию в MS Excel. При копировании формулы адрес автоматически изменяется относительно позиции столбца и строки новой ячейки | Например: A9, D25, F5 | Полезна, когда нужно повторить один и тот же расчет на несколько столбцов и строк |
| **Абсолютная** (Absolute reference) | Ссылка на ячейку остается неизменной при копировании или заполнении. Знак **$** фиксирует ссылку | Например: **$A$1**. Знак $ ставится как перед именем столбца, так и перед номером строки. Клавиша **F4** позволяет быстро изменить адрес на абсолютный | Используется, когда адрес ячейки должен быть постоянным (например, при умножении на фиксированный коэффициент) |
| **Смешанная** (Mixed address) | Знак **$** стоит либо перед именем столбца, либо перед номером строки, фиксируя только часть адреса | Например: $А9 или А$7 | Комбинированное использование |
| **Связывание файлов** | Абсолютные и относительные ссылки можно использовать при одновременной работе с несколькими файлами | Пример ссылки на внешний файл: **'=[1.xlsx]Лист1!$D$4'** | Позволяет отражать изменения, внесенные в одном файле, на данных другого файла |

## V. Автоматизация вычислений и ввода

### 5.1. Автозаполнение (Autocomplete)

**Автозаполнение** — удобная функция для автоматического заполнения таблицы данными и выполнения вычислений с помощью формул.

| Случай использования | Описание |
| :--- | :--- |
| **Одинаковые данные** | Ввод одного и того же значения в несколько смежных ячеек |
| **Последовательность** | Заполнение строки/столбца данными с заданным промежутком (например, четные числа, дни недели, даты, названия месяцев) |
| **Формулы** | Применение одной формулы к смежным ячейкам для быстрого вычисления |

**Маркер автозаполнения** — маленький квадрат в правом нижнем углу активной ячейки.

### 5.2. Автоматическое суммирование

**Автоматическое суммирование** — используется кнопка Σ для сложения чисел в ячейках.

## VI. Встроенные функции

### 6.1. Общие сведения о функциях

**Функции в MS Excel** — вычислительные операции, проводимые по определенному алгоритму или формулам.

- **Назначение:** MS Excel имеет более 200 встроенных функций для решения простейших и сложных задач
- **Ввод:** Функции можно выбрать из **Библиотеки функций** (команда *Формулы => Библиотека функций*) или с помощью кнопки **fx** (Вставить функцию)
- **Аргументы:** Могут быть числа, выражения, адрес ячейки или адрес диапазона. Аргументы можно набрать на клавиатуре или указать, выделив ячейки/диапазоны на рабочем листе
- **Категории:** Финансовые, логические, текстовые, дата и время, ссылки и массивы, математические, статистические, инженерные, аналитические, проверка свойств и значений
- **Сложные функции:** Создаются путем вложения одной функции в другую (используются круглые скобки), например: `=ОКРУГЛ(СРЗНАЧ(H4:H8); 2)`

### 6.2. Основные функции

| Категория | Функция | Назначение | Примеры |
| :--- | :--- | :--- | :--- |
| **Математические** | СУММ | Вычисляет сумму всех чисел | `=СУММ(5; 4; 2; 8; 9)` |
| | КОРЕНЬ | Вычисляет квадратный корень | `=КОРЕНЬ(54289)` |
| | СТЕПЕНЬ | Возводит число в указанную степень | `=СТЕПЕНЬ(54289; 2)` |
| **Статистические** | СРЗНАЧ | Определяет среднее значение | `=СРЗНАЧ(G4:G6)` |
| | МАКС | Определяет максимум среди аргументов | `=МАКС(D4:D10)` |
| | МИН | Определяет минимум среди аргументов | `=МИН(D4:D10)` |
| | СЧЕТЕСЛИ | Подсчитывает количество непустых ячеек в диапазоне, удовлетворяющих условию | |
| **Логические** | ЕСЛИ (IF) | Проверяет логическое выражение. Возвращает *значение 1*, если условие истинно, иначе — *значение 2* | `=ЕСЛИ(A1<20000; 12; 15)` |
| | И (AND) | Принимает ИСТИНА, если все аргументы ИСТИННЫ | |
| | ИЛИ (OR) | Принимает ИСТИНА, если хотя бы один аргумент ИСТИНЕН | |


### 6.3. Функции для работы с данными

| Функция | Назначение | Синтаксис |
| :--- | :--- | :--- |
| **ПОИСКПОЗ** (MATCH) | Ищет заданное значение в строке или столбце и возвращает ее **порядковый номер** (начиная с первой позиции заданного диапазона) | `ПОИСКПОЗ(искомое_значение, просматриваемый_массив, [тип_сопоставления])` |
| **ИНДЕКС** (INDEX) | Возвращает значение или ссылку на значение из таблицы/диапазона, заданного **номером строки** и **номером столбца** | `ИНДЕКС(массив, номер_строки, номер_столбца)` |
| **ПРЕДСКАЗ** (FORECAST) | Используется для вычисления прогноза (значение линейного тренда или проекции по линейному приближению) | |
| **КОРРЕЛ** (CORRELATION) | Определяет корреляционную зависимость между точками данных (R). Если R > 0, связь положительно возрастающая; если R < 0, связь отрицательно убывающая | |
| **СМЕЩ** (OFFSET) | Используется для определения диапазона ячеек со смещением | |

## VII. Условное форматирование

**Условное форматирование** — инструмент для визуального анализа данных. Позволяет раскрасить рабочий лист так, чтобы каждый цвет соответствовал определенным данным.

**Доступ:** Кнопка **Условное форматирование** на панели **Стили** ленты **Главная**.

| Вид форматирования | Назначение |
| :--- | :--- |
| **Правила выделения ячеек** | Форматирование ячеек, отвечающих определенным условиям (Больше, Меньше, Между, Равно, Текст содержит и др.) |
| **Правила отбора первых и последних значений** | Выделение верхних или нижних значений в диапазоне (например, первые 5 элементов) |
| **Гистограмма** | Отображение горизонтального столбца в каждой ячейке (частичная заливка), помогающая рассмотреть значения относительно других |
| **Цветовые шкалы** | Сравнение диапазона значений с использованием градации двух или трех цветов |
| **Наборы значков** | Вставка значков (например, стрелок или светофоров), определяемых на основе сравнения значений ячеек |
| **Управление правилами** | Дает возможность удалять правила или вносить изменения |

## VIII. Графическое представление данных (Диаграммы)

### 8.1. Общие сведения о диаграммах

**Диаграммы** используются для представления множества числовых данных в графическом формате, что упрощает их понимание.

| Характеристика | Описание |
| :--- | :--- |
| **Построение** | На вкладке **Вставка** выбрать **Диаграммы** |
| **Типы** | Гистограмма, График, Круговая, Линейчатая, С областями, Точечная и др. |
| **Смешанная диаграмма** | Создается путем совмещения разных типов диаграмм |
| **Настройка** | В меню **Макет** (панель **Подписи**) можно изменить название диаграммы, добавить названия вертикальных/горизонтальных осей, добавить легенду, создать таблицу данных |
| **Легенда** | Подпись ряда произвольных данных в столбцах и строках на диаграмме |

### 8.2. Построение графиков и линий тренда

**Построение графика функции:**
1. Выделить данные
2. Выполнить последовательность команд: *Вставка => Диаграмма Точечная => Точечная с гладкими кривыми*

**Добавление линии тренда:**
1. Выделить точку на диаграмме
2. В контекстном меню выбрать *«Добавить линию тренда...»*
3. Выбрать нужный вид тренда (например, *«Линейная»*)
4. Можно вывести на диаграмму уравнение зависимости и значение R²

**Виды линий тренда:**
- Линейный
- Логарифмический
- Полиномиальный
- Степенный
- Скользящий средний
- Экспоненциальный

**Коэффициент R-квадрат (R²)** — величина достоверности, определяющая уровень точности прогноза. При значении, равном или близком к единице, данные подходят для прогноза.

## IX. MS Excel как база данных

### 9.1. Структура базы данных в Excel

Excel может использоваться в качестве **базы данных (БД)**.

| Элемент БД | Соответствие в таблице Excel | Определение |
| :--- | :--- | :--- |
| **Строки (Записи)** | Строки таблицы | Содержат данные об объекте |
| **Поля** | Столбцы таблицы | Содержат какое-либо свойство объекта |
| **Список (Таблица)** | Простая таблица в рабочей области | Первоначальная база данных в Excel |
| **Диапазон базы данных** | Таблица, выполняющая функцию БД | Расположена в рабочей области листа; включает строку заголовка с именами полей |

### 9.2. Требования к рабочему листу Excel для использования в качестве БД

1. Каждому столбцу должно соответствовать только **одно поле**
2. Поля должны располагаться **одно за другим без пробелов**
3. Заголовки полей должны отображаться в **первой строке** (не должны повторяться)
4. Заголовок поля должен размещаться в **одну ячейку**
5. Записи должны располагаться **непосредственно под заголовком**
6. Записи ведутся последовательно, **без пустых строк**. Пустая строка понимается как **конец базы данных**
7. Все записи БД должны располагаться на **одном рабочем листе**
8. На листе может быть несколько БД, но **только одна может быть активной**

### 9.3. Инструменты для работы с БД (Диалоговое окно «Форма»)

Инструменты для работы с БД представлены в разделе меню **«Данные»**.

Для вызова формы нужно добавить команду **«Форма»** на **Панель быстрого доступа** (через «Другие команды» — «Команды не на ленте»).

| Кнопка/Функция | Назначение |
| :--- | :--- |
| Окно «Форма» | Показывает номер листа, количество полей и записей. Текущая запись |
| **Добавить** | Добавляет записи в БД |
| **Удалить** | Удаляет записи |
| **Вернуть** | Возвращает удаленную запись |
| **Далее/Назад** | Перемещение вперед и назад по записям |
| **Критерии** | Используется для поиска данных по заданному условию |
| **Закрыть** | Завершение работы с БД |

## X. Поиск и замена информации

### 10.1. Команда «Найти и выделить»

- **Команда:** Главная ► Редактирование ► Найти и выделить ► Найти или **Ctrl+F**
- Позволяет искать: **текст**, **формулы**, **заметки** и **выделенные ячейки**
- Поиск и замена осуществляются через диалоговое окно
- Поиск происходит только в пределах **выбранного диапазона** ячеек

### 10.2. Символы подстановки (Wildcards)

- **?** — заменяет один символ (например, `3?7` находит все трехзначные числа, начинающиеся с 3 и заканчивающиеся 7)
- **\*** — заменяет любое количество символов (например, `5*` находит все числа, начинающиеся с 5)
- *Ограничение:* Нельзя найти числовые значения определенного формата (например, `$54.00`) если вводить только число (`$5`)

### 10.3. Вспомогательные функции для поиска данных

**Проверка данных (Валидация):**
- Команда: Данные — Работа с данными — Проверка данных
- Используется для создания **Списка** в ячейке, чтобы выбрать название города или продукта

**Функция ПОИСКПОЗ (MATCH):**
- **Назначение:** Ищет заданное значение в строке или столбце и возвращает ее **порядковый номер**
- **Синтаксис:** `ПОИСКПОЗ(искомое_значение, просматриваемый_массив, [тип_сопоставления])`
- **Тип сопоставления:** 0 (точное соответствие), 1 (наибольшее значение), -1 (наименьшее значение)

**Функция ИНДЕКС (INDEX):**
- **Назначение:** Возвращает значение или ссылку на значение из таблицы/диапазона
- **Синтаксис:** `ИНДЕКС(массив, номер_строки, номер_столбца)`

## XI. Сортировка данных

**Сортировка** — отображение данных в определенном порядке.

**Команда:** Данные — Сортировка и фильтр

### 11.1. Типы сортировки

1. **Простая сортировка:** По возрастанию (от А до Я) или по убыванию (от Я до А)
2. **Сложная сортировка:** Выполняется по нескольким полям одновременно

### 11.2. Особенности сортировки

- Сортировка может выполняться по одному или нескольким столбцам
- При сортировке по одному столбцу Excel предлагает:
  1. **Автоматически расширить выделенный диапазон** (рекомендовано для полной сортировки по всем столбцам)
  2. Сортировать в пределах указанного выделения
- **Диалоговое окно «Сортировка»** позволяет сортировать данные по:
  - **Значению**
  - **Цвету ячейки**
  - **Цвету шрифта**

## XII. Фильтрация данных

**Фильтрация** — способ найти данные в списке в соответствии с конкретными условиями.

### 12.1. Основные характеристики фильтрации

- В отличие от сортировки, фильтрация **не нарушает порядок записей**
- Отображаются только строки, соответствующие условиям; ненужные строки скрываются
- Отфильтрованные строки можно редактировать, форматировать, печатать или использовать для построения диаграмм
- Отфильтрованные элементы отображаются **синим цветом**
- **Удаление фильтра:** Повторно выделить диапазон и нажать кнопку «Фильтр»

### 12.2. Типы фильтрации

1. **Фильтр (Автофильтр):** Используется при простых условиях фильтрации
2. **Расширенный фильтр:** Фильтрация по сложным условиям

### 12.3. Правила работы с Автофильтром

- Работает только в **непрерывном диапазоне**
- Самая верхняя строка считается **заголовком** (к ней фильтрация не применяется)
- Для отбора чисел используется команда **«Числовые фильтры»** (условия: равно, больше, между, первые 10, выше среднего и др.)
- Условия можно составлять с операторами **«И»** или **«ИЛИ»**

### 12.4. Расширенный фильтр

- Используется для **составных (сложных) условий**
- **Преимущества:** Возможность добавить **несколько критериев** и сохранить отфильтрованные записи **в другой таблице**
- **Команда:** Данные > Фильтр > Дополнительный

**Параметры диалогового окна «Расширенный фильтр»:**
- **Обработка:** «Фильтровать список на месте» или **«Скопировать результат в другое место»**
- **Исходный диапазон:** Выбирается диапазон фильтра
- **Диапазон условий:** Область, где прописываются критерии фильтрации (например, Длина > 200 км)
- **Поместить результат в диапазон:** Указывается место для размещения отфильтрованных данных

## XIII. Анализ данных и прогнозирование

### 13.1. Динамические ряды

**Динамические ряды** — ряды, отражающие развитие событий в хронологическом порядке (например, изменение температуры, пропускной способности сети, анализ продаж).

**Прогноз (прогнозирование)** — предсказание будущего с помощью научных методов, научная модель предстоящих событий.

### 13.2. Методы прогнозирования в Excel

**1. Линия тренда (Trend Line)**
- График функции, используемый для составления прогнозов на основе имеющихся табличных данных
- **Линейный тренд** используется, когда значения увеличиваются или убывают с постоянной скоростью

**2. Коэффициент R-квадрат (R²)**
- Величина достоверности, определяющая уровень точности прогноза
- При значении, равном или близком к единице, данные подходят для прогноза

**3. Функция ПРЕДСКАЗ (FORECAST)**
- Относится к статистической категории
- Используется для вычисления прогноза (значение линейного тренда или проекции по линейному приближению)

**4. Функция КОРРЕЛ (CORRELATION)**
- Определяет корреляционную зависимость между точками данных (R)
- Если R > 0, связь положительно возрастающая
- Если R < 0, связь отрицательно убывающая

## XIV. Моделирование в Excel

MS Excel выделяется простотой и легкостью для **математического и теоретического моделирования** на компьютере.

### 14.1. Этапы моделирования (применительно к Excel)

1. **Постановка задачи:** Определение цели, начальных данных и анализ объекта
2. **Построение информационной модели (Формализация):** Приведение свойств объекта к выбранной форме
3. **Разработка компьютерной модели:** Выбор программной среды (например, MS Excel). Алгоритм зависит от выбранной программы. В Excel рабочая область может быть спланирована с разделами "Данные", "Решение", "Математическая модель", "Ответ"
4. **Компьютерный эксперимент:** Тестирование модели и проведение исследования (проверка правильности построения модели)

### 14.2. Применение Excel для моделирования нейронных сетей

- Проектирование нейронной сети может осуществляться в MS Excel
- Табличная модель нейронной сети, созданная в Excel, может быть **легко преобразована и расширена**
- Используются встроенные функции Excel для вычисления выходного сигнала нейрона

| Сфера применения | Примеры задач, решаемых в Excel | Используемые инструменты |
| :--- | :--- | :--- |
| **Математика** | Построение графиков функций и поиск точек их пересечения | Диаграммы, формулы |
| **Физика** | Построение графиков зависимости физических величин (например, силы тока от сопротивления по закону Ома) | Абсолютная адресация, формулы, графики |

## Глоссарий терминов

**Диапазон** — таблица, которая выполняет функцию базы данных и расположена в рабочей области листа Excel.

**Индекс** — функция, которая возвращает значение или ссылку на значение из таблицы или поименованного диапазона, заданного номером строки и номером столбца.

**Фильтрация** — способ найти данные в списке в соответствии с конкретными условиями.

**Электронная таблица (spreadsheet)** — интерактивная система обработки данных, заданных в виде таблицы.

**Ячейка** — основная единица хранения данных в электронной таблице, прямоугольник на пересечении столбцов и строк.

**Книга** — основной файл MS Excel, состоящий из нескольких листов.

**Лист** — отдельная страница в книге Excel, содержащая таблицу данных.

**Формула** — выражение, начинающееся со знака =, которое выполняет вычисления.

**Функция** — предопределенная формула, выполняющая вычисления по определенному алгоритму.

**Легенда** — подпись ряда произвольных данных в столбцах и строках на диаграмме.

**Линия тренда** — график функции, используемый для составления прогнозов на основе имеющихся табличных данных.

**Автозаполнение** — функция для автоматического заполнения таблицы данными и выполнения вычислений с помощью формул.

**Условное форматирование** — инструмент для визуального анализа данных путем раскрашивания ячеек в зависимости от их значений.
