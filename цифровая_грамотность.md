# ЦИФРОВАЯ ГРАМОТНОСТЬ

## ЦИФРОВИЗАЦИЯ В КАЗАХСТАНЕ

### Преимущества цифровизации
*   Обеспечение доступности и продвижения товаров/услуг (государственных, коммерческих, мирового масштаба).
*   Появление систем управления, замещающих человека (например, для определенных категорий предприятий).
*   Улучшение понимания потребителей и повышение качества продуктов/услуг.
*   Создание новых продуктов/услуг и повышение гибкости предложений.
*   Перевод бумажных документов (листки нетрудоспособности, трудовые книжки и др.) в электронный формат.

### Риски и пути решения
*   Внедрение цифровой экономики является серьезным испытанием для экономики Казахстана.
*   Основные риски связаны с вопросами **национальной безопасности** и **конкурентоспособности** на мировом рынке.
*   Отставание от развитых стран в темпах цифровизации может привести к выпадению из научно-технического прогресса.
*   **Пути решения:** Цифровизация традиционных отраслей экономики, развитие цифровой инфраструктуры, цифровизация госорганов, изменение модели производства через развитие предпринимательства в сфере цифровых технологий.

## ЭЛЕКТРОННОЕ ПРАВИТЕЛЬСТВО

### Портал eGov.kz
*   **eGov.kz** — портал "Электронное правительство", единый механизм взаимодействия государства с гражданами и между госорганами с помощью ИТ.
*   Позволяет получить более 400 видов автоматизированных государственных услуг онлайн, сокращая очереди.
*   Работает на казахском, русском и английском языках.
*   Для доступа к большинству услуг необходима авторизация через ЭЦП.

### Основные разделы eGov.kz
Портал включает следующие основные разделы:

*   **Услуги:** Позволяет получать онлайн различные справки и документы.
*   **Платежи:** Позволяет осуществлять онлайн-платежи в бюджет, оплачивать коммунальные услуги и др..
*   **Открытое правительство (5 компонентов):** Обеспечивает прозрачность госсектора и привлекает граждан к принятию государственных решений. Включает:
    *   **«Открытые данные»:** Площадка с общедоступными и бесплатными данными госорганов.
    *   **«Открытые НПА» (Нормативно-правовые акты):** Платформа для публикации и обсуждения проектов НПА.
    *   **«Открытые бюджеты»:** Создана для упрощения понимания бюджета и развития общественного контроля за расходованием бюджетных средств.
    *   **«Открытый диалог»:** Площадка для обратной связи между госорганами и обществом (включает блоги акимов).
*   **Личный кабинет:** Позволяет просматривать различные сведения и результаты оказанных услуг.

## ТЕХНОЛОГИЯ BLOCKCHAIN

### Основные понятия и принцип работы
*   **Блокчейн** – это способ структурирования данных. Представляет собой **распределенную базу данных** (разнесенную систему), где хранилище данных не связано с общим процессором.
*   Блокчейн — это упорядоченный список записей, называемых **блоками**.
*   Каждый блок дополняется информацией и имеет **временную метку** и **ссылку на предыдущий блок**.
*   Для контроля сохранения важных записей используется система шифрования, или **криптография**.
*   **Криптография** — математические методы, позволяющие сохранять информацию в тайне и подтверждать личность. Блокчейн использует криптографические ключи для защиты (например, при вводе номера банковской карты в онлайн-магазине).
*   **Цифровые ключи:**
    *   **Один ключ (закрытый)** подтверждает, что вы можете ввести данные о транзакции.
    *   **Другой ключ (публичный/открытый)** позволяет другим подтвердить, что транзакция введена вами.

### Применение
*   **Устранение посредников:** Технология устраняет необходимость в финансовых учреждениях (банках) для перевода денег, беря на себя учет транзакций, установление личности и заключение контрактов.
*   **Умные контракты:** Контракты, которые выполняются автоматически, когда стороны вводят свои ключи (подписывают договор).
*   База данных может включать не только единицы стоимости (биткоин), но и любой цифровой контент, включая программный код.
*   Примеры применения: автоматизация процесса выставления счетов за электроэнергию, создание реестров недвижимости.

### Что такое эргономика?

| Концепт | Описание | Источники |
| :--- | :--- | :--- |
| **Определение** | **Эргономика** (от греч. *ergon* – «работа», *nomos* – «закон») – наука, посвященная всестороннему исследованию трудового процесса, обеспечению безопасности и удобства для человека в системе «человек – технология – окружающая среда». | |
| **Цель** | Предоставление рекомендаций по повышению эффективности и комфорта рабочей среды. | |

| **Компьютерная эргономика** | Наука, занимающаяся изучением взаимоотношений человека и компьютера. | |

Ноги должны касаться пола, угол в колене – 90°. **Монитор:** Расположение на расстоянии 60–70 см от глаз (не ближе 50 см), на уровне глаз или чуть ниже. **Освещение:** Естественное освещение оптимально должно падать слева. | |

### Проблемы интернет-зависимости в обществе

| Концепт | Описание | Источники |
| :--- | :--- | :--- |
| **Определение** | **Интернет-зависимость** – психическое расстройство, проявляющееся в неспособности пользователя вовремя и добровольно выйти из сети, а также навязчивом стремлении подключиться к ней вновь. | |
| **Причины зависимости** | Восприятие интернета как хобби, нерациональное использование социальных сетей, сетевые/онлайн-игры, просмотр фильмов. | |
| **Виды зависимости** | **Веб-серфинг** (бесконечный поиск информации), стремление к **виртуальным знакомствам и переписке**, **азартные онлайн-игры** и ненужные покупки/аукционы. | |
| **Последствия** | Ухудшение здоровья, снижение мышления, культуры речи, грамотности. Физические последствия: нарушение зрения, заболевания позвоночника (сколиоз, остеохондроз), ухудшение работы сердца и сосудов (аритмия, гипертония). | |
| **Профилактика** | Контролировать время в интернете, избегать онлайн-игр, использовать сеть как онлайн-библиотеку для углубления знаний, проводить время со сверстниками, гулять и играть в подвижные игры. | |
