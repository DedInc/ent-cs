## Внутреннее устройство компьютера и периферийные устройства

### Конфигурация компьютера
*   Персональный компьютер – универсальная техническая система. Основная конфигурация: **системный блок, монитор, клавиатура**.
*   **Системный блок** считается основным устройством.
    *   **Внутренние устройства:** Расположены внутри системного блока.
    *   **Внешние (Периферийные) устройства:** Присоединяются к нему снаружи.

### Внутренние устройства системного блока

1.  **Материнская плата (Motherboard):** Основа компьютера, печатная плата, к которой подключаются все остальные элементы. На ней расположены: процессор, ОЗУ, чипсет, BIOS, кэш-память, шины, слоты расширения.
2.  **Процессор (Микропроцессор):** Электронная микросхема, управляющая работой каждого устройства, выполняющая вычисления и обработку информации.
3.  **Оперативная память (ОЗУ)**.
4.  **Жесткий диск**.
5.  **Привод оптических дисков** (CD и DVD).
6.  **Видеоадаптер (Видеокарта):** Устройство для вывода графической информации на экран монитора, преобразует данные в видеосигнал. Использует видеопамять (VRAM).
7.  **Звуковая карта:** Позволяет создавать и записывать высококачественный звук.
8.  **Сетевая плата (Сетевая карта):** Позволяет соединять компьютер с компьютерной сетью (например, Ethernet, Token Ring, Wi-Fi).
9.  **Порты ввода-вывода:** Разъемы на корпусе для подсоединения устройств.
10. **Блок питания:** Вторичный источник электропитания. Преобразует сетевое напряжение до требуемых значений (обычно 5 или 12 В). Мощность от 150 до 320 Вт.

### Ключевые компоненты на Материнской плате

#### Шина (Bus)
*   **Шина** – подсистема для передачи данных между функциональными блоками.
*   **Виды шин:**
    *   **Шина данных** – передает данные между компонентами.
    *   **Адресная шина** – передает адреса памяти и устройств.
    *   **Командная шина** – передает управляющие сигналы.
*   **Типы шин:** ISA, MCA, EISA, SCSI, VLB, PCI, AGP, USB и т.д.

#### Чипсет (Chipset)
*   Набор микросхем, обеспечивающих взаимодействие ЦП с памятью, устройствами ввода-вывода и адаптерами.

#### BIOS (Basic Input Output System)
*   **BIOS** – базовая система ввода-вывода.
*   **Функции:**
    *   Проверка компонентов при загрузке (POST – Power-On Self-Test).
    *   Настройка устройств.
    *   Загрузка операционной системы.
    *   Взаимодействие компонентов.
*   **Современный вариант:** UEFI (Unified Extensible Firmware Interface).

#### Сопроцессор
*   Расширяет возможности ЦП, ускоряя выполнение арифметических операций и обработку видеоизображений.

#### Слот расширения
*   Разъем для подключения дополнительных плат (видеокарты, звуковые, сетевые).

## Процессор и его характеристики

### Определение
*   **Центральный процессор (CPU)** – это «мозг» компьютера, его основная микросхема.
*   Быстродействие компьютера напрямую зависит от скорости процессора.
*   **Основные производители:** Intel и AMD.
*   **Многоядерные процессоры** (два и более ядра) повышают производительность за счет разделения данных.

### Основные параметры процессора

1.  **Разрядность процессора:**
    *   Величина, определяющая количество информации (в битах), которую ЦП способен обработать **за один такт**.
    *   Примеры: 8-разрядный, 16-разрядный, 32-разрядный, 64-разрядный процессор.
    *   Чем выше разрядность, тем быстрее обработка данных.

2.  **Тактовая частота (Frequency):**
    *   Количество элементарных операций (тактов), выполняемых процессором **за единицу времени**.
    *   Измеряется в Герцах (Гц, МГц, ГГц).
    *   Чем выше тактовая частота, тем быстрее работает процессор.

3.  **Кэш-память:**
    *   Быстродействующая промежуточная память малого объема.
    *   Используется для временного хранения копий данных из ОЗУ.
    *   **Порядок доступа к данным:** Регистры → Кэш-память → Оперативная память (ОЗУ) → Жесткий диск (HDD).

### Устройство ядра процессора

*   **Арифметико-логическое устройство (АЛУ):** Выполняет математические и логические операции.
*   **Устройство управления:** Обеспечивает последовательность выполнения команд.
*   **Регистры:** Ячейки памяти внутри процессора для временного хранения данных.
*   **Сопроцессор:** Устройство, расширяющее вычислительные возможности (например, для обработки чисел с плавающей запятой).
*   **Процессорный интерфейс:** Разъем на материнской плате для размещения процессора.
*   **Системная шина:** Информационная магистраль, обеспечивающая связь всех устройств.

## Глоссарий терминов устройств компьютера

**Видеокарта** - плата, через которую монитор подключается к компьютеру.

**Жесткий диск** - внешняя память компьютера, предназначенная для постоянного хранения данных, программ операционной системы и часто используемых пакетов программ.

**Конфигурация** - компоновка системы с четким определением характера, количества, взаимосвязей и основных характеристик ее функциональных элементов; совокупность аппаратных средств и соединений между ними.

**Материнская плата** (от англ. motherboard) - сложная многослойная печатная плата, являющаяся основой построения вычислительной системы (компьютера).

**Монитор** - устройство визуального представления данных.

**Оптический привод** - устройство для считывания данных с оптического носителя.

**Персональный компьютер** - это настольная или переносная ЭВМ, удовлетворяющая требованиям общедоступности и универсальности применения.

**Процессор** - основное устройство ЭВМ, предназначено для выполнения вычислений по хранящейся в запоминающем устройстве программе и обеспечения общего управления ЭВМ.

**Системный блок** - основной узел компьютера, внутри которого установлены наиболее важные компоненты: материнская плата с процессором, жесткий диск, дисковод гибких дисков, дисковод компакт-дисков.

#### огические основы компьютера (Архитектура фон Неймана)

| Принцип Архитектуры Фон Неймана (1945) | Суть |
| :--- | :--- |
| **1. Принцип двоичного кодирования** | Вся информация (данные и команды) в компьютере хранится в двоичном коде (0 и 1). |
| **2. Принцип однородности памяти** | Коды команд и данных хранятся в одной и той же памяти. Компьютер не различает их, пока они не используются. Над командами можно выполнять действия, как над данными. |
| **3. Принцип адресности** | Память состоит из пронумерованных ячеек (адресов). Процессор имеет произвольный доступ к любой ячейке. |
| **4. Принцип программного управления** | Задачи выполняются с помощью алгоритмов, представленных в виде последовательности команд (программы). |

**Основные устройства процессора (ЦПУ)**:
*   **Арифметико-логическое устройство (АЛУ):** Выполняет арифметические и логические преобразования.
*   **Устройство управления (УУ):** Формирует и подает сигналы управления для выполнения операций.
*   **Функции процессора:** Выбор (команды из памяти), Декодирование (перевод на язык Ассемблер), Выполнение (операций) и Управление (передача результата в память).
*   **Блок регистров (Локальная память):** Временное хранение информации управления. Увеличивает скорость процессора.

## Виртуальные машины (ВМ)

**Определение и назначение:**

*   **Виртуальная машина (ВМ)** — это программное обеспечение, эмулирующее (создающее) виртуальные компьютеры внутри основной (хостовой) операционной системы (ОС).
*   ВМ используются для решения различных задач и востребованы как в бытовых, так и в промышленных целях.
*   **Виртуальная операционная система** — это ОС, установленная на виртуальной машине.

**Архитектура гипервизоров (программ для управления ВМ):**

Гипервизоры делятся на две основные архитектуры:

| Архитектура | Ключевые особенности | Преимущества и недостатки | Пример |
| :--- | :--- | :--- | :--- |
| **Монолитная** | Драйверы устройств подключаются к гипервизору. | **Недостаток:** Потенциальное снижение безопасности, так как уязвимости в коде драйверов (выполняющихся в пространстве гипервизора) могут позволить получить контроль над хостовой и гостевыми ОС. | VMware ESX |
| **Микроядерная** | Драйверы устройств работают внутри хостовой ОС (которая запускается как "базовая партиция" в виртуальной среде). | **Преимущества:** Не требуются специальные драйверы для гипервизора; совместимость с любым оборудованием, имеющим драйверы в базовой ОС. Более высокий уровень безопасности, так как в гипервизоре отсутствует сторонний код. | Hyper-V |

**Программное обеспечение для создания ВМ:**

1.  **VirtualBox (Oracle):**
    *   Бесплатное, с открытым исходным кодом.
    *   Поддерживает гостевые ОС: Mac OS Х, Linux, Android, Solaris, Open Solaris, OS / 2, OpenBSD и др..
    *   Простое в использовании, подходит для домашнего использования.
2.  **VMware Workstation Pro (VMware):**
    *   Проприетарное (платное) ПО.
    *   Поддерживает более 200 ОС (Windows, macOS, Linux, Android и др.).
    *   Предназначено для профессионального использования и интеграции с сервисами VMware.
    *   Эмулирует все основные устройства компьютера, включая BIOS, что отличает его от Virtual Box.
    *   Существует бесплатная версия для личного пользования — VMware Workstation Player.
3.  **Hyper-V:**
    *   Встроенный компонент ОС Windows 7-10 (кроме начальной и домашней редакций).

**Основные элементы работы с VirtualBox (Практикум):**

*   Интерфейс VirtualBox включает список ВМ, панель инструментов (для запуска, настроек), отображение параметров ВМ и статусов.
*   Настройки позволяют задать папку для файлов ВМ, язык, обновления. Дополнительные **плагины** (например, Oracle_VM_Virtual Box_Extension_Pack) расширяют возможности программы.
*   Для установки гостевой ОС необходим ее образ (например, файл в формате .iso).
*   Установка гостевой ОС на ВМ (например, Ubuntu) аналогична установке на обычный компьютер.

## Характеристики мобильных устройств (Смартфоны и планшеты)

**Планшет** — это устройство с сенсорным экраном диагональю 7" и более, выполняющее роль компьютера.

| Характеристика | Описание | Применительно к планшетам (примеры) |
| :--- | :--- | :--- |
| **Процессор** | Основа вычислительной мощности. | Может иметь 1-8 ядер, рабочая частота 1–2 ГГц. |
| **Графический процессор (GPU)** | Модуль, отвечающий за задачи, связанные с графикой. Важен для ресурсоемкого графического контента, такого как 3D игры. | Мощный видеопроцессор важен для "тяжелой" графики. |
| **Оперативная память (ОЗУ/RAM)** | Определяет общую скорость работы системы. Чем больше ОЗУ, тем быстрее работает устройство, особенно с многозадачностью и ресурсоемкими приложениями. | От 512 МБ до 3 ГБ. |
| **Встроенная память** | Объем памяти, доступный без сменных карт. Работает быстрее, чем сменные носители, но дороже. | Аналог жесткого диска, обычно от 4 ГБ до 128 ГБ. |
| **Автономность работы** | Зависит от емкости аккумулятора, которая измеряется в миллиампер-часах (мА·ч), например, 4000 мА·ч. | Работа связана с емкостью аккумулятора. |
| **Операционная система (ОС)** | Программная платформа устройства. Каждая ОС имеет магазины приложений для расширения возможностей устройства. | **Смартфоны:** Android 5.x-9.х, Windows Phone, Apple iOS. **Планшеты:** Android 5.x-9.х, Windows 8.x-10.х, Apple iOS. |
| **Технологии экрана** | Используемые технологии: LCD (SLCD), TFT (IPS), Amoled. | **TFT:** Имеют ограниченный угол обзора. **IPS:** Улучшенная версия TFT, решает проблему углов обзора. **AMOLED:** Экономят энергию и имеют самые высокие углы обзора. |
| **Беспроводная связь** | Позволяет обмениваться данными и выходить в сеть. | **Wi-Fi:** Для доступа в Интернет (десятки метров). **Bluetooth:** Для обмена данными (несколько метров). **3G, 4G:** Модули подключения к Интернету. |
| **Материалы корпуса** | Четыре основных типа материалов: | Пластик (поликарбонат), металл (алюминий, сталь, магний), стекло (алюмосиликатное, например, Corning), экзотические материалы (дерево, кожа). |

**Основные направления развития программного обеспечения мобильных устройств:**

*   Стандартизация.
*   Объектно-ориентированное программирование.
*   Интеллектуализация пользовательского интерфейса.
*   Универсализация отдельных модулей прикладных программ.
*   Расширение возможностей программ и программных систем.
*   Использование облачных технологий для совместной работы.

### История развития вычислительной техники

| Концепт | Описание | Источники |
| :--- | :--- | :--- |
| **Ранние изобретения** | **1623 г.:** Простейшая счетная машина (сложение/вычитание) создана Вильгельмом Шиккардом. **1642 г.:** **«Паскалево колесо»** (первая вычислительная машина с сохраняемой в памяти программой) создана Блезом Паскалем. | |
| **Аналитическая машина** | **1834 г.:** Изобретена Чарльзом Бэббиджем (считается отцом современного компьютера). Состояла из 4 частей: **склад** (память), **мельница** (арифметическое устройство), **блок управления** и **блок ввода/вывода**. | |
| **Первый программист** | Ада Лавлейс, написавшая программу для машины Бэббиджа. | |
| **Первый электронный ПК** | **1946 г.:** **ENIAC** (США). Вес 30 тонн, 18 тыс. электронных ламп, скорость 5000 операций/сек. | |
| **Поколения ЭВМ (4 поколения)** | **I (1946–1954):** Электронные лампы, 10–20 тыс. операций/сек, пульт управления, перфоленты. **II (1955–1964):** Полупроводники (транзисторы), 100–500 тыс. операций/сек, перфокарты/перфоленты. **III (1965–1974):** Интегральные микросхемы (до 1000 транзисторов), 1 млн операций/сек, алфавитно-цифровой терминал, развитые ОС (мультипрограммирование). **IV (С 1975 г. по наст. время):** Большие интегральные схемы, 10–100 млн операций/сек, возникновение понятия **«персональный компьютер» (ПК)**. | |
| **Компьютеры будущего** | Развитие техники приведет к созданию **искусственного интеллекта**. Прогнозируются: **Биокомпьютер** (функционирует как живой организм на основе клеток); **Молекулярный компьютер** (увеличение памяти в сотни раз); **Квантовый (фотонный) компьютер** (использование фотонов как носителей информации, прогнозируется ускорение работы в сотни раз). | |

### Как работает компьютер

| Концепт | Описание | Источники |
| :--- | :--- | :--- |
| **Архитектура компьютера** | Составные части компьютера и взаимосвязь между ними (функциональная схема). В основе лежит **магистрально-модульная структура**. | |
| **Магистраль (шина данных)** | Набор электронных линий, связывающих процессор, память и внешние (периферийные) устройства. Состоит из шины данных, шины адресов и шины управления. | |
| **Основные внутренние устройства** | **Процессор (CPU):** «Мозг» компьютера, выполняет все вычисления. **Материнская плата (Mother board):** Основная плата, содержит шины, соединяющие внутренние устройства. **Оперативная память (RAM):** Временное хранение данных и программ, данные удаляются при выключении. **Жесткий диск (Внешняя память):** Длительное хранение данных. **Видеоадаптер:** Передает изображение на монитор в виде видеосигнала. **Сетевой адаптер:** Связывает компьютер с сетью или Интернетом. | |
| **Компьютерные порты** | Электронное устройство (с разъемами), предназначенное для обмена данными между ПК и внешними устройствами. **Внутренние порты** (магистральные шины) – для внутренних устройств (жесткие диски). **Внешние порты** – для внешних устройств (принтеры, сканеры), располагаются на задней панели. | |

### Операционные системы

| Концепт | Описание | Источники |
| :--- | :--- | :--- |
| **Определение ОС** | **Операционная система (ОС)** – самое важное программное обеспечение. Выступает как «мостик» между пользователем и компьютером. | |
| **Основные функции** | Управляет памятью, процессором, всеми программами и аппаратным обеспечением. Организует согласованное выполнение всех процессов, хранение информации, запуск задач | |
